Adiciona a funcionalidade de listar as redes sociais mais utilizadas sem ter que alterar o seu layout e adicionar funções no template.

== Description ==

Este plugin tem como funcionalidade adicionar a funcionalidade de share para as principais e mais utilizadas redes sociais.

Na versão 1.0 conta com as redes sociais:
	* "Twitter"
	* "Facebook"
	* "Google Plus"
	* "Google Buzz"
	
Atualizado para encurtador migre.me
	
== Installation ==

Após a ativação do plugin, acesse o Dash Board e vá em CHW Plugins e configure como achar melhor.

Ative / Desative quais redes você não quer que apareça e configure o CSS de acordo com o seu site.

Observe o arquivo mythumb.gif, por padrão é ele que o facebook irá reconhecer como imagem no "Like Button", então favor
alterar para um de seu gosto.
